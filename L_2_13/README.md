# Утилита My-Cut

Реализация утилиты `cut` на языке Go, аналогичной стандартной команде Unix `cut`. Утилита считывает входные данные из STDIN или файла, разбивает каждую строку по заданному разделителю и выводит указанные поля (колонки).

## Флаги

- **`-f fields`** - номера полей для вывода (обязательный флаг)
  - Формат: `1,3,5-7,10` (отдельные номера и диапазоны)
  - Пример: `-f 1,3-5` выведет 1-й и 3-5 столбцы
  
- **`-d delimiter`** - разделитель полей (по умолчанию: табуляция `\t`)
  - Пример: `-d ","` для CSV файлов
  
- **`-s`** - флаг "separated" (optional)
  - Если указан, игнорирует строки без разделителя
  - Пример: `-s` выведет только строки, содержащие разделитель

## Примеры использования

### Базовое использование с табуляцией (по умолчанию)
```bash
echo -e "one\ttwo\tthree\nfour\tfive\tsix" | go run main.go -f 1,3
```
Вывод:
```
one	three
four	six
```

### Использование диапазонов
```bash
echo -e "a\tb\tc\td\te" | go run main.go -f 2-4
```
Вывод:
```
b	c	d
```

### Использование пользовательского разделителя
```bash
echo "name,age,city" | go run main.go -f 1,3 -d ","
```
Вывод:
```
name,city
```

### Использование флага -s (только строки с разделителем)
```bash
printf "one\ttwo\nno_separator\nalpha\tbeta" | go run main.go -f 1 -s
```
Вывод:
```
one
alpha
```

### Обработка файла
```bash
go run main.go -f 1,3-5 ./test_data.txt
```

### Запуск тестов

```bash
# Все тесты
go test -v

# С измерением покрытия
go test -v -cover

go test -bench=. -benchmem
```

### Пример работы с большим файлом

```bash
# Генерируем большой CSV файл
seq 1 10000000 | awk '{print NR","$1*2","$1*3}' > large.csv

# Извлекаем 1-й и 3-й столбцы
time go run main.go -f 1,3 -d "," large.csv > output.csv
```
